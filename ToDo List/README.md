# To Do List App

A simple and clean To Do List web app where you can add and delete tasks — built using HTML, CSS, and JavaScript.

---

## About This Project

This is one of my first JavaScript projects. I built it to practice DOM manipulation — creating elements dynamically, handling button click events, and removing items from the page.

---

## Features

- Type a task and click **Add** to add it to the list
- Click **Delete** next to any task to remove it
- Clean and minimal design

---

## Files

| File | Purpose |
|------|---------|
| `ToDoList.html` | Page structure |
| `ToDoList.css` | Styling and layout |
| `ToDoList.js` | Add and delete task logic |

---

## How to Run

1. Download or clone this repository
2. Open the `ToDo List` folder
3. Double-click `ToDoList.html`
4. Opens directly in your browser — no setup needed

---

## What I Learned

- DOM manipulation (`createElement`, `appendChild`, `remove`)
- Event listeners (`addEventListener`)
- Dynamically adding and deleting elements on a webpage
